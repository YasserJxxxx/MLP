Objetivo del Proyecto y Criterios de Éxito

Objetivo Principal

Desarrollar y entrenar un modelo MLP robusto para clasificar correctamente las tres especies de Iris (setosa, versicolor, virginica).

Se basa en las cuatro medidas físicas de la flor (sépalo y pétalo).

Criterios de Éxito

  • Técnico: Exactitud (Accuracy) en prueba mayor o igual a 95%.
  • Analítico: F1-Score Ponderado mayor o igual a 0.95.
  • Despliegue: Modelo y Preprocesador (Escalador) guardados funcionalmente.

Preparación de Datos (Fases 2 y 3)

Dataset IRIS

Consta de 150 muestras distribuidas en 3 clases, con 50 muestras por cada especie (perfectamente balanceado).

Transformaciones Clave

Variables escaladas con StandardScaler y etiquetas codificadas a One-Hot Encoding.

La estandarización mejora la convergencia del MLP.

División Estratificada

Dataset dividido manteniendo la proporción de clases (estratificado):

Entrenamiento: 70%, Validación: 15%, Prueba: 15%.

Evaluación y Métricas Finales

Rendimiento en Conjunto de Prueba

Accuracy: 95.65% (0.96)

F1-Score Ponderado: Aprox. 0.96

El modelo clasificó correctamente 22 de las 23 muestras, superando ambos criterios de éxito ($\ge 0.95$).

Error Concentrado

  • Setosa (Clase 0): Rendimiento perfecto (100% de métricas).
  • Único Error (1/23): Un caso real de versicolor (Clase 1) fue clasificado incorrectamente como virginica (Clase 2).
  • Causa: Confusión habitual entre versicolor y virginica debido al solapamiento de sus características.

Despliegue y Artefactos (Fase 6)

Artefactos para Producción

Se exportaron dos componentes críticos para garantizar la funcionalidad fuera del entorno de entrenamiento:

  • Modelo MLP: iris_mlp_model.h5.
  • Escalador: iris_scaler.pkl.

El Escalador es obligatorio para estandarizar nuevos datos antes de la predicción.

Proceso de Predicción (Flujo)

  1. 1. Entrada de 4 medidas.
  2. 2. Preprocesamiento: Aplicación de scaler.transform().
  3. 3. Inferencia: model.predict() (genera vector de probabilidades).
  4. 4. Resultado: Mapeo a nombre legible (ej. "virginica") y confianza.
